# Portal: Site

## site:logs
Shows list of log files.

#### Access
service view

#### Request
```
$ curl "$PORTAL_API_URL/v1/site/{id}/log" \
  -H "Authorization: Bearer $PORTAL_API_KEY" \
  -H "Accept: application/json"
```
