# Portal: Site

## site:sso-url
Get Sso Url.

#### Access
service view

#### Input
- integer `id` (required): System ID of the sso url.

#### Request
```
$ curl "$PORTAL_API_URL/v1/site/{id}/sso-url" \
  -H "Authorization: Bearer $PORTAL_API_KEY" \
  -H "Accept: application/json"
```
