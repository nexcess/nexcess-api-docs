Portal: Ssh Keys
----------------

**since** 0.0.0

ssh-key:show
============

Shows details of an Ssh Key.

**Endpoint**:  GET /v1/ssh-key/{id}

**Access**: logged-in users

**Parameters**: none

**Request**:
```
curl -i "$PORTAL_API_URL/v1/ssh-key/$KEY_ID" \
  -H "Authorization: Bearer $PORTAL_API_KEY" \
  -H "Accept: application/json"
```
